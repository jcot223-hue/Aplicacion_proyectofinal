*Aplicacion* 
-
Descripcion: Esta aplicacion fue realizada en flutter que su funcion es recibir reportes y guardando todos los datos recibidos mandandolos a una api.

*Codigo de la aplicaion*
-
1.Flutter

2.Http

3.JSON

4.Shared_preferences

5.Path_provider

*Funcionalidades principales*
-
-Usuario

1.Registro

2.Iniciar sesion

3.Crear perfil

4.Guardar los datos del perfin

-Subir un reporte

1.Crear un reporte

2.Abjuntar una foto

3.Escribir un texto

4.Listado de reportes mandados

5.Guardar o eliminar un datos en la api

6.Eliminar un reporte

*Repositorio de GitHub*
-
1.Repositorio de la aplicacion: https://github.com/jcot223-hue/Aplicacion_proyectofinal/tree/main

2.Abir el visual del archivo

3. Abrir consola con "ctrl+j" y escribir "flutter pub get" y "flutter run" para ejecutar la aplicacion

(Antes de ejecutar la aplicacion tienes que ejecutar la api)
